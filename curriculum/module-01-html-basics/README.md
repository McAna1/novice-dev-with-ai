# Module 1 — What is HTML?

**Duration:** 2–3 class sessions  
**Prerequisites:** None  
**Next module:** Module 2 — Semantic HTML5

---

## What Students Learn

- What HTML is and what problem it solves
- The three layers of a webpage (HTML, CSS, JavaScript)
- Tags, elements, and attributes
- The required HTML5 boilerplate
- Head vs Body — hidden vs visible content
- Opening a webpage in Chrome and using View Source
- Using VS Code and Live Server for the first time
